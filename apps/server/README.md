# Backend Server (Deprecated)

**Note:** This subproject has been deprecated since it cannot work properly. In development or production environments, please use the Python backend in `python-server` instead.

Express + TypeScript + Prisma + MySQL skeleton.

## Usage

- dev: `npm run dev`
- build: `npm run build`
- start: `npm run start`
- prisma generate: `npm run prisma:generate`
- prisma migrate: `npm run prisma:migrate`
